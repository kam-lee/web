Research Focus
CFD Software Development

Hypersonic aircraft face extremely complex external airflow environments and extreme high temperature and pressure working conditions, accompanied by strong thermodynamic, chemical, and electromagnetic non-equilibrium effects, which place extremely high demands on the accuracy and efficiency of numerical simulation technology. The widely popular commercial computing software in the industrial sector almost lacks the ability to simulate such flows. The widely open source code is not designed for such high-speed flows, and secondary development inevitably inherits a large number of redundant functions, making it difficult to get started and costly to learn. Based on years of technical practice and accumulation, HPL has integrated various technical resources to develop a universal simulation analysis software called Hurricane for rockets and ramjet engines, and has opened up some of the code to OpenHurricane.

OpenHurricane is an open-source library written in C++ 17 and CUDA for computing supersonic reacting flows. In the current version, the OpenHurricane is able to compute chemistry source terms and gas physical properties on the CPU and GPU platforms alternatively. OpenHurricane is in its development phase, and there would be more features in it.
